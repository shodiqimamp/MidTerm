## Table of Contents

## Database Structure

This Project Have 3 Collection, videos, comments, and products.

### Videos Collection
```
{
  title: String
  thumbnailUrl: String
  youtubeUrl: String
  productId: String
  comments: String
},
```

### Comments Collection
```
{
  name: String
  comment: String
  createdAt: Date
  videoId: String
}
```
### Products Collection
```
 {
  title: String
  price: Number
  link: String
}
```
